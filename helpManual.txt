add task for floating task : add <task title>
add task for event task with starting date and end date : add <task title> -on <day> <month> <start time> to <end time><am/pm>
add task for task that have a deadline : add <task title> -by <day> <month> <time><am/pm>
add task for daily recurrent task: repeat <task title> -start <day> <month> <start time> to <end time><am/pm> -every <number> day -until <day> <month> <year>
add task for weekly recurrent task: repeat <task title> -start <day> <month> <start time> to <end time><am/pm> -every <number> week on <days> -until <day> <month> <year>
add task for monthly recurrent task: repeat <task title> -start <day> <month> <start time> to <end time><am/pm> -every <number> month -until <day> <month>
add task for yearly recurrent task: repeat <task title> -start <day> <month> <start time> to <end time> <am/pm> -every <number> year

delete task for floating task : delete <index>
delete task for event task with starting date and end date : delete <index>
delete task for task that have a deadline : delete <index>

update a deadline task: update <index> <task title> -by <day> <month> <time><am/pm>

update a weekly recurrent task: repeat <task title> -start <day> <month> <start time> to <end time><am/pm> -every <number> week on <days> -until <day> <month> <year>
update a monthly recurrent task: repeat <task title> -start <day> <month> <start time> to <end time><am/pm> -every <number> month -until <day> <month>
update a yearly recurrent task: repeat <task title> -start <day> <month> <start time> to <end time> <am/pm> -every <number> year
